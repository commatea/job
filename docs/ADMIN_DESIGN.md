# SpecLab 관리자 페이지 상세 설계서

**버전**: 1.0
**작성일**: 2025-12-25

---

## 1. 개요

### 1.1 목적
SpecLab 관리자 페이지는 플랫폼의 콘텐츠, 사용자, 시스템을 효율적으로 관리하기 위한 백오피스 시스템입니다.

### 1.2 관리자 역할 구분

| 역할 | 권한 | 설명 |
|------|------|------|
| **SUPER_ADMIN** | 모든 권한 | 시스템 설정, 관리자 계정 관리 |
| **ADMIN** | 콘텐츠/사용자 관리 | 자격증, 직업, 게시물, 사용자 관리 |
| **MODERATOR** | 콘텐츠 검수 | 게시물 승인, 신고 처리 |
| **ANALYST** | 읽기 전용 | 통계 조회만 가능 |

```
POST   /api/v1/admin/auth/login          # 관리자 로그인
POST   /api/v1/admin/auth/logout         # 로그아웃
POST   /api/v1/admin/auth/refresh        # 토큰 갱신
POST   /api/v1/admin/auth/2fa/verify     # 2FA 인증
```

### 5.2 대시보드
```
GET    /api/v1/admin/dashboard/stats     # 핵심 통계
GET    /api/v1/admin/dashboard/alerts    # 알림 목록
GET    /api/v1/admin/dashboard/pending   # 처리 대기 항목
```

### 5.3 자격증 관리
```
GET    /api/v1/admin/certifications                    # 목록
POST   /api/v1/admin/certifications                    # 등록
GET    /api/v1/admin/certifications/{id}               # 상세
PUT    /api/v1/admin/certifications/{id}               # 수정
DELETE /api/v1/admin/certifications/{id}               # 삭제
PATCH  /api/v1/admin/certifications/{id}/status        # 상태 변경
POST   /api/v1/admin/certifications/{id}/prerequisites # 선수 자격증 연결
POST   /api/v1/admin/certifications/{id}/careers       # 직업 연결
POST   /api/v1/admin/certifications/import             # 일괄 등록
GET    /api/v1/admin/certifications/export             # 내보내기
```

### 5.4 사용자 관리
```
GET    /api/v1/admin/users                       # 목록
GET    /api/v1/admin/users/{id}                  # 상세
PATCH  /api/v1/admin/users/{id}/status           # 상태 변경 (정지/해제)
DELETE /api/v1/admin/users/{id}                  # 삭제
GET    /api/v1/admin/users/{id}/activities       # 활동 이력
POST   /api/v1/admin/users/{id}/password-reset   # 비밀번호 초기화
POST   /api/v1/admin/users/{id}/force-logout     # 강제 로그아웃
```

### 5.5 콘텐츠 관리
```
GET    /api/v1/admin/posts                       # 게시물 목록

PATCH  /api/v1/admin/posts/{id}/reject           # 반려
DELETE /api/v1/admin/posts/{id}                  # 삭제

GET    /api/v1/admin/reports                     # 신고 목록
```

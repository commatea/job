# SpecLab 플랫폼 기획서

**프로젝트명**: SpecLab - 자격증 기반 미래 스펙 네비게이션 플랫폼
**버전**: 1.0
**작성일**: 2025-12-25

---

## 1. 플랫폼 개요

### 1.1 SpecLab 이란?
SpecLab은 **자격증을 토대로 직업, 창업에 대한 정보를 제공**하고, 사용자의 **미래 스펙**을 체계적으로 관리할 수 있는 올인원 커리어 플랫폼입니다.

### 1.2 핵심 가치
| 가치 | 설명 |
|------|------|
| **Spec Navigation** | 게임 스킬트리처럼 직관적인 자격증 로드맵 시각화 |
| **Career Intelligence** | AI 기반 맞춤형 커리어 컨설팅 |
| **Startup Guide** | 창업에 필요한 자격증 및 법적 요건 안내 |
| **Community** | 합격 후기, 스터디 모집, 정보 공유 생태계 |

### 1.3 타겟 사용자
- **취업 준비생**: 원하는 직무에 필요한 자격증 탐색
- **직장인**: 이직/승진을 위한 추가 자격증 취득
- **창업 희망자**: 창업에 필요한 필수 자격증 및 법적 요건 확인
- **학생**: 진로 탐색 및 스펙 빌딩 계획 수립

## 3. 페이즈별 개발 로드맵

### Phase 1: MVP - 기반 구축 (현재)
> **목표**: 핵심 기능 구현 및 서비스 검증

#### 3.1.1 주요 기능
| 기능 | 설명 | 우선순위 |
|------|------|----------|
| **자격증 테크트리** | React Flow 기반 시각화, 드래그/줌 지원 | ⭐⭐⭐ |
| **자격증 상세 정보** | 시험 일정, 응시 자격, 합격률, 수수료 | ⭐⭐⭐ |
| **직업 연결** | 자격증 → 취업 가능 직종 연결 | ⭐⭐⭐ |
| **회원 가입/로그인** | 이메일 + 소셜 로그인 (카카오, 구글) | ⭐⭐⭐ |
| **반응형 UI** | 모바일/태블릿 완벽 지원 | ⭐⭐⭐ |
| **관리자 CRUD** | 자격증/직업 데이터 관리 | ⭐⭐ |

#### 3.1.2 기술 구현 사항
```
Frontend:
├── 랜딩 페이지 (SpecLab 브랜딩)
├── 테크트리 뷰어
│   ├── 카테고리 필터 (IT, 건설, 전기, 의료 등)
│   ├── 노드 클릭 → 상세 정보 사이드패널
│   └── 레벨별 레이어링 (기능사→산업기사→기사→기술사)
├── 자격증 검색 페이지
├── 직업 정보 페이지
├── 창업 가이드 페이지
├── 마이페이지
│   ├── 보유 자격증 관리
│   ├── 관심 자격증 북마크
│   └── 취득 목표 설정
└── 관리자 대시보드

Backend:
├── 인증 API (JWT + Refresh Token)
├── 자격증 CRUD API
├── 직업/창업 CRUD API
├── 사용자 프로필 API
└── 검색 API
```

## 8. 보안 설계

### 8.1 인증/인가
- **JWT 기반 인증**: Access Token (15분) + Refresh Token (7일)
- **역할 기반 접근 제어 (RBAC)**: USER, ADMIN, SUPER_ADMIN
- **소셜 로그인**: OAuth 2.0 (카카오, 구글)

### 8.2 데이터 보안
- **HTTPS 강제**: 모든 통신 암호화
- **비밀번호 해싱**: bcrypt (cost factor: 12)
- **개인정보 암호화**: AES-256 (이메일, 전화번호)
- **SQL Injection 방지**: SQLAlchemy ORM 파라미터 바인딩

### 8.3 API 보안
- **Rate Limiting**: IP당 100req/분, 사용자당 300req/분
- **CORS 설정**: 허용된 도메인만 접근
- **Input Validation**: Pydantic 스키마 검증
